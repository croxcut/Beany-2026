Beany 2026

- Areact native implementation of Beany 2025 from Kotlin : )*

### Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app
   
   **Windows/Linux**
   ```bash
   npx expo start
   ```

   **WSL2 - you need to enable tunnel**
   ```bash
   npx expo start -c --tunnel
   ```
